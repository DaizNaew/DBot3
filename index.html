Welcome to DaizNaew's repo of Dbot3.

Dbot started as a side project to learn java, and grew into a minor work project to integrate java with certain elements of web, irc and standard xml.

Hope you do enjoy this as a versatile tool for irc managing, or for an understanding in using java generally or irc oriented.
